## 单体模式

单体模式的思想在于：保证一个特定的类仅有一个实例

在 js 中没有类，只有对象，当创建一个新对象的时候，实际上已经没有其他对象与其类似，因此新对象已经是单体了

```js
var obj = {
    myprop: "my value"
}
```

在 js 中，对象之间永远不会完全相等（除非它们是同一个对象），因此即使创建一个具有完全相同成员的同类对象，它也不会与第一个对象完全相同

```js
var obj2 = {
    myprop: "my value"
}

obj == obj2;  // false
obj === obj2;  // false
```
